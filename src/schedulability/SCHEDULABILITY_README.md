# Schedulability Experiment - README

## Overview
This experiment evaluates schedulability of real-time task sets with victim tasks under offset scheduling, implementing equations from the offset scheduling paper.

## Files

### Python Script
- **`schedulability_fnal.py`** - Main experiment script
  - Generates task sets using RandFixedSum (Dirichlet method)
  - Tests schedulability with HP/MP/LP victim tasks
  - Implements Equations 12, 14, 15 (victim) and 17 (non-victims)
  - Saves results to CSV file

### MATLAB Script
- **`plot_schedulability_results.m`** - Visualization script
  - Reads CSV file generated by Python script
  - Creates publication-quality plots
  - Displays statistics table
  - Saves figures in multiple formats

### Helper Script
- **`run_matlab_helper.py`** - MATLAB integration helper
  - Checks if MATLAB is available in PATH
  - Runs MATLAB scripts from Python
  - Can be used standalone or imported

### Output Files
- **`schedulability_results.csv`** - Experiment data
  - Columns: Util_Range, Util_Midpoint, HP_Schedulability, MP_Schedulability, LP_Schedulability
- **`schedulability_plot.png`** - Standard resolution plot
- **`schedulability_plot_hires.png`** - High resolution (300 DPI) plot
- **`schedulability_plot.fig`** - MATLAB figure file (editable)

## Usage

### Step 1: Run Python Experiment
```bash
python schedulability_fnal.py
```

This will:
- Generate 100 task sets per utilization range (default)
- Test schedulability with victims from HP, MP, and LP groups
- Save results to `schedulability_results.csv`
- Display a matplotlib plot
- **Ask if you want to run MATLAB script automatically**

If you answer 'y', the Python script will automatically execute the MATLAB script and display results.

### Step 2 (Optional): Plot in MATLAB Manually
If you didn't run MATLAB automatically, you can run it manually:
```matlab
plot_schedulability_results
```

This will:
- Read `schedulability_results.csv`
- Generate plots showing HP > MP > LP schedulability
- Display statistics in console
- Save figures in multiple formats

### Alternative: Run MATLAB Helper Standalone
```bash
python run_matlab_helper.py
```

This helper script can:
- Check if MATLAB is available in system PATH
- Run the MATLAB plotting script
- Provide detailed error messages if MATLAB is not found

## Parameters (in `schedulability_fnal.py`)

You can edit these parameters at the top of the script:
```python
P_S = [5, 10, 20, 50, 100, 200, 1000]  # Period set
E_MIN, E_MAX = 1, 50                    # Execution time range
NUM_SETS_PER_GROUP = 1000               # Task sets per utilization group
```

In the main section:
```python
n_tasks = 11  # Number of tasks per set
util_ranges = [(0.02,0.08), (0.12,0.18), ...]  # Utilization ranges
```

## Expected Results

The plots should show:
- **HP (Blue line)**: Highest schedulability (victim has fewer higher-priority tasks)
- **MP (Green line)**: Medium schedulability
- **LP (Red line)**: Lowest schedulability (victim has most higher-priority tasks)

Schedulability decreases as utilization increases.

## Requirements

### Python
- numpy
- pandas
- matplotlib

Install with:
```bash
pip install numpy pandas matplotlib
```

### MATLAB
- Base MATLAB (tested on R2019a and later)
- No additional toolboxes required
- **For automatic execution from Python**: MATLAB must be in system PATH

### Adding MATLAB to Windows PATH:
1. Search for "Environment Variables" in Start Menu
2. Click "Edit the system environment variables"
3. Click "Environment Variables" button
4. Under "System variables", find and select "Path"
5. Click "Edit"
6. Click "New" and add MATLAB bin directory, e.g.:
   ```
   C:\Program Files\MATLAB\R2023a\bin
   ```
7. Click "OK" to save
8. Restart your terminal/command prompt

## Notes

- The experiment implements Rate-Monotonic Priority Assignment (RMPA)
- Tasks are sorted by period (smaller period = higher priority)
- Delta_peak is calculated as: Δ_peak = D_v - R_v
- Schedulability requires: Δ_peak > 0 AND all lower-priority tasks schedulable
